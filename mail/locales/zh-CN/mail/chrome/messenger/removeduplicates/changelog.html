22.05.2012: Version 0.1.13

  • Bugfix: Double click did not work with latest Thunderbird versions anymore. As an alternative it is now possible to use a middle click to toggle the message state between DEL and KEEP.

  • Bugfix: Code cleanup and compatibility changes for Thunderbird 13 API.

07.03.2012: Version 0.1.11

  • Bugfix: Comparison of message body is working again. Now even better then before. During comparison also attachements are regarde. Unfortunately this option is very time consuming.
  • Bugfix: A little bit more duplicates are found now.
  • Feature: Compare recipient list. This is useful to remove duplicate sent messages.
  • Feature: Added an option to show the duplicate message dilog even if no duplicates were found.
  • Feature: Added an option to compare messages by date (not time).
  • Feature: Brazilian localisation: pt-BR.
