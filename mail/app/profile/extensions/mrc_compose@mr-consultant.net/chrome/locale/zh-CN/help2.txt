<h4>Show nb recipients if above</h4>
The number of recipients is indicated on the left of the label of each field.<br/>
It is shown only if above that number.

<h4>Suggested max nb of recipients</h4>
For the fields "To" and "Cc", a warning is shown if the number of recipients <br/>is above that number.<br/>
The warning tells the user it's better to use the field "Bcc" instead.

<h4>Start search after nb characters</h4>
The search starts after this number of characters entered.<br/>
On slow machines, it's better to choose "2" or "3".
